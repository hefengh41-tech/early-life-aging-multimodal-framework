# early-life-aging-multimodal-framework
Multimodal prediction of biological aging using DNA methylation, EHR-derived features, and machine learning

## Pipeline Overview

1. DNA methylation preprocessing
2. EHR processing + LLM feature extraction
3. Multi-modal data integration
4. Machine learning prediction of biological aging
5. SHAP-based interpretability
6. Ablation study for modality contribution
